The configurations for the pipeline:
- 1. Install all the packages in the VICTRE following the VICTRE pipeline
- 2. Configure the directory to the random seed file in the helper_function.py
- 3. Configure the directory to the phantom generation command and phantom configuration templates in the phantom_gen.py
- 4. Configure the directory to the phantom compression command, cropping commands, and Febio software in the phantom_compress_crop.py
- 5. Configure the directory to the output dataset in the run_pipeline.py
- 6. run the phantom generation script: python run_pipeline.py
